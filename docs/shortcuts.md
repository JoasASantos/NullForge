# Keyboard Shortcuts

> `Cmd` = `⌘` on macOS, `Ctrl` on Linux/Windows

## Global

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+P` | Command Palette |
| `Cmd+,` | Settings |
| `Cmd+Shift+?` | Keyboard Shortcut Cheat Sheet |
| `Cmd+Shift+L` | Layout Preset Picker |
| `Cmd+Shift+M` | Toggle Editor Maximize |
| `Escape` | Close active modal/palette |

## Panel Toggles

| Shortcut | Action |
|----------|--------|
| `Cmd+B` | Toggle Sidebar |
| `Cmd+Shift+B` | Toggle AI Panel |
| `Cmd+J` | Toggle Bottom Panel |
| `Cmd+\`` | Open Shell (bottom panel) |

## Focus Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+2` | Focus Shell |
| `Cmd+3` | Open / focus Debugger |
| `Cmd+4` | Open / focus Disassembler |
| `Cmd+5` | Focus AI Panel |

## Tool Panels

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+D` | Open Disassembler |
| `Cmd+Shift+G` | Open Debugger |
| `Cmd+Shift+N` | Open Network Monitor |
| `Cmd+Shift+K` | Open Shellcode Generator |

## Editor

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Inline AI at cursor |
| `Cmd+W` | Close active tab |
| `Cmd+1` … `Cmd+9` | Go to tab N |
| `Cmd+Option+→` | Next tab |
| `Cmd+Option+←` | Previous tab |

## Debugger

| Shortcut | Action |
|----------|--------|
| `F5` | Continue |
| `F6` | Pause |
| `Shift+F5` | Stop |
| `F9` | Toggle breakpoint |
| `F10` | Step over |
| `F11` | Step into |
| `Shift+F11` | Step out |

## ROP / Disassembler

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+R` | ROP gadget search |
| `Cmd+Shift+E` | Export ROP chain |

## Workspace

| Shortcut | Action |
|----------|--------|
